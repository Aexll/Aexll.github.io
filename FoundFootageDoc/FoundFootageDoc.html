What is Found Footage Camera Effects ?

Found Footage Camera Effects is a pack of multiple post process effect to create a “VHS” effect on cameras that look like found footage videos you can find online

Utilisations

The Asset contains 6 differents Post Process Volumes Blueprints

Each of them are a presset to demonstrate multiple usages

Each Blueprints contains a PostProcess

The Rendering Features > Post Process Materials > Array contains all the post process materials applied by the blueprint, you can add or remove effects to your needs

The differents effects are detailed in the next section ‘Content’

Content

Inside the 6 blueprints, there are 8 differents bases post process effects

each of them are used as an instance, to easely change the parameters

The 6 bases effects are:

Bands

Deform
Sharp

emulate a “bad compression” effect

VHS

VHS (custom depth)

Just like VHS but only applie to object that are rendered in the Custom Depth Pass

TL;DR : Only on applied on some objects

Toons
Zoom

! be sure to have the Zoom Post Process Material in the Rendering Features > Post Process Materials > Array !

Add the ability to Zoom the camera (while keeping the default resolution, unlike the zoom on a camera actor)

You can use the SetScalarParameterValue Node to change the zooming amount (1 is default, 2 mean zoomed 2x)

it also contains a Widget Retainer Material Effect , (a post process effect that works on UI elements)

Requirments

Lumen and virtual shadow map are enabled in every demo, disabling them may change the end result

Technical Documentation

About Multiplayer

The Pack should be multiplayer ready.


Like any post process, if you want to apply the effect only to a given person, you will need to copy the
Rendering Features > Post Process Materials > Array in the camera settings of the player

The Zoom factor (in the material collection) is NOT replicated (don’t worry its a good thing), it mean that if you change it on the client side, the zoom will only apply to that client

Limitations

Exposure Metering Mode is set to ‘manual’ on every post process since it was causing problems with some effects, if you find that the lights are “propagating” on your whole screen its probably that the autoexposure is set to Auto Exposure Histogram and need to be switched back to Manual
